JavaScript for the Web

JavaScript is how you build interactivity on the web. Take this class to learn how to work with elements, make AJAX calls, paint on a canvas, and much more!

  • Lesson #1

    Introduction to JavaScript

    This lesson is the start of the Javascript class. Get setup and ready to start writing code for the web!

  • Lesson #2

    Adding JavaScript to your Page

    Add JavaScript to your page using the three available ways, external, internal and inline.

  • Lesson #3

    Declaring Variables and Constants

    Variables and constants hold your data. Learn how to declare them in this lesson.

  • Lesson #4

    Primitive Data Types

    This lesson will go over all the primitive data types that are supported by JavaScript.

  • Lesson #5

    Math: Operators, Properties and Methods

    Learn about how to do arithmetic and work with the properties and methods provided in the Math library!

  • Lesson #6

    Logic and Conditionals

    Add logic and use conditionals in your code to allow your script to make decisions.

  • Lesson #7

    Loops: For loop, While loop, Do While loop, For In loop

    Shorten your code by working with all the loops like for loops, while loops and do while loops.

  • Lesson #8

    Functions: Reuse your Code

    Reuse the JavaScript code you write by putting it inside a function!

  • Lesson #9

    Arrays: Lists of Data

    Arrays are a list of data that you can do many things with like iterate over and manipulate.

  • Lesson #10

    Objects: Properties and Methods

    Objects in JavaScript represent a collection of properties and methods belong to the same thing.

  • Lesson #11

    Classes: Constructors, Inheritance, Super

    Classes in JavaScript provide a blueprint for the creation of objects.

  • Lesson #12

    Sets: The Unique Values Collection

    Sets are collections of elements that only allow unique values.

  • Lesson #13

    Maps: The Key Values Collection

    Maps are collections of key-value pairs that act as a dictionary.

  • Lesson #14

    Timeouts and Intervals

    Timeouts and intervals allow you to control when and how often a function is invoked.

  • Lesson #15

    Date: Work with Moments in Time

    Express dates and times using JavaScript's built-in Date object!

  • Lesson #16

    Document Object Model

    The Document Object Model describes how pages are structured and how you can alter elements.

  • Lesson #17

    Elements: Classes, Properties, Attributes, Styles

    Using JavaScript, you can manipulate an elements classes, properties, attributes and styles.

  • Lesson #18

    Browser Events and Event Listeners

    Learn how to listen in to events triggered by the user and execute code when they occur!

  • Lesson #19

    JSON: The Language of Data Transfer

    JSON is a data transfer language that powers the web you know today.

  • Lesson #20

    AJAX and Fetch

    AJAX helps deliver powerful and real-time experiences on the web.

  • Lesson #21

    Cookies: Storing Data on the Client

    Cookies let you store small pieces of information on the client, instead of the server.

  • Lesson #22

    Local Storage and Session Storage

    Local Storage and Session Storage let you store information across sessions or for just a single session.

  • Lesson #23

    Errors, Exceptions, and Debugging

    As with any programming language, you will need to learn how to deal with errors and debug!

  • Lesson #24

    JavaScript Conclusion and Wrap Up

    This lesson is the conclusion and wrap up of this class on JavaScript!
